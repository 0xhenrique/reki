Yet another CLI program to fetch system information.

** Installation

Clone this repo:
 #+begin_src 
git clone --depth=1 git@github.com:henrique-marques-vsoft/reki.git
 #+end_src

Have [[Leiningen][https://leiningen.org/]] installed in your machine and then run:
 #+begin_src 
   lein run
 #+end_src
 
Shrimple as that.
